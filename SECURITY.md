# Security Policy

This repo is **capstone curriculum** (Markdown + a reference-solution submodule).
The runnable code lives in the [`jaricsng/shopkit`](https://github.com/jaricsng/shopkit)
submodule — see its `SECURITY.md` and `SECURITY-FINDINGS.md` for the application's
security posture and accepted teaching tradeoffs.

## Reporting

For a problem in the **curriculum** (a module, a command, a doc): open an issue
or PR. For a vulnerability in the **reference application**: report it privately
on the shopkit repo (GitHub Security → *Report a vulnerability*), not here.

Do not paste real credentials, tokens, or customer data into issues or PRs.
